Welcome to Astroberry OS

Astroberry OS is an operating system for Raspberry Pi for controlling astronomy equipment.

  • Set wifi regulatory domain using raspi-config before connecting to an access point.
  • Connect your Raspberry Pi to a local network or home wifi. Hotspot connection limits certain functionalities, which require Internet connection.
  • Communication with Astroberry OS is protected with a unique security certificate. Install and trust the certificate in your browser to avoid security warnings.
  • Remember, that Astroberry OS is a free and open-source operating system, made possible by your support.
